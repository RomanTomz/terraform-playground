# Chapter 04 - CI/CD

## Mini lecture (sports analogy)
CI/CD is your assistant coaching staff.
They review your tactics before match day and stop obvious mistakes.

Terraform intent here:
- run automatic checks in pull requests,
- preview planned changes,
- keep apply manual for safety and cost control.
