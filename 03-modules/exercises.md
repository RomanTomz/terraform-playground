# Hands-on Exercises

These exercises focus on **intent + practice**.

## Exercise 1: Module reuse
Intent: Run dev and prod with different bucket names using same module source.

Suggested commands:
```bash
terraform init
terraform plan
terraform apply
terraform destroy
```

## Exercise 2: Tag strategy
Intent: Add a new tag variable and pass different values per environment.

Suggested commands:
```bash
terraform init

terraform destroy
```
