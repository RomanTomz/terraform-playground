# Hands-on Exercises

These exercises focus on **intent + practice**.

## Exercise 1: Terraform lifecycle drill
Intent: Run local-file and observe plan/apply/destroy output.

Suggested commands:
```bash
terraform init
terraform plan
terraform apply
terraform destroy
```

## Exercise 2: Variable tuning
Intent: Change project_name and environments in terraform.tfvars and compare outputs.

Suggested commands:
```bash
terraform init
terraform plan
terraform apply
terraform destroy
```

## Exercise 3: Predict before plan
Intent: Write what you expect in named_environments, then run plan/apply.

Suggested commands:
```bash
terraform init
terraform plan
terraform apply
terraform destroy
```
